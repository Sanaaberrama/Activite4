PARTIE 2 : RESUME DU CHAPITRE 2

1/ Reseau informatique :

Un reseau informatique est un ensemble d equipement interconnectes pour permettre la communication de donnees entre les applications.

Le r seau informatique s appuie essentiellement sur deux notions fondamentales qui sont l interconnexion et la communication.

2/ Les caracteristiques de base d un reseau :

Un reseau est caracterise par :

  • la topologie (qui definit l architecture d un reseau)

  • le debit exprime en bits/s

  • la distance maximale qui depend de la technologie mise en oeuvre

  • nombre maximal de noeuds que l on peut interconnecter.

3/Terminologie en reseau :

  • Station de travail : c est toutes les machines capables de d envoyer des donnees vers un reseau

  • Noeud : c est toute entite d information pouvant etre envoye sur le reseau

  • Paquet : c est la plus petite unite d information pouvant etre envoye sur le reseau.

  • Reseaux heterogenes : le reseau heterogene est un reseau d ordinateurs relies entre eux par l Ethernet.

  • Le debit: est represente par la vitesse de connexion.

4/ Elements d un reseau :

Le reseau est constitue par les ordinateurs, les logiciels, les supports de LAN, les equipements d interconnexion du LAN.

5/Classification des reseaux :

Les reseaux peuvent etre classifies en fonction des diffrents criteres suivants :

    a/ Par l envergure (LAN, MAN, WAN)

  • LAN : Reseau local d entreprise.

  • MAN : Reseau metropolitain ou urbain.

  • WAN : Reseau grande distance.

    b/ L ouverture :

  • Intranet (Reseau local d entreprise)
  • Extranet (Reseau d entreprise ouvert vers l exterieur)
  • internet (Reseau d entreprise ouvert vers l interieur)

    c/ Les supports de connexion :

  • Il existe deux types de supports de connexion (filaire et sans fils)
  • Reseaux filaires (fibre optique, reseau Telephonique)
  • Reseaux sans fils (ondes radios, bluetooth)

    d/ Le type d organisation :


Il existe deux types d organisation de reseaux

  • Les Reseaux d egale a egale (c est les reseaux ou chaque ordinateur est libre de partager ses ressources).
  • Les reseaux clients/serveurs (c est les reseaux ou la plupart des stations " clients " sont regit par des administrateurs a travers leurs stations)

6/ Les types de topologie :

Il existe plusieurs types de topologies qui sont :

  • a) Les reseaux en bus : les machines sont reliees par un cable coaxial (le bus) et chaque ordinateur est connecter sur le bus par un connecteurs BNC.
  • b) Les reseaux en etoile : toutes les stations sont connectees a un commutateur en envoyant les donnees vers ce dernier.
  • c) Les reseaux en anneau : les ordinateurs sont situes sur une boucle et communiquent chacun a leur tour, les donnees transitent de station en station en suivant l anneau qui regenere le signale a chaque fois.
  • d) Les reseaux mailles : les reseaux mailles utilisent plusieurs chemins de transferts entre differents noeuds, cette structure reprend un cablage en etoile regroupant differents noeuds de reseaux.

PARTIE 3 :RESUME DU CHAPITRE 3

1/ Les Composants materiel d un Reseau informatique.

  • Les cartes reseau :

    • Ce sont des composants materiels qui permettent a un ordinateur de se connecter a un reseau , elles sont installees a l interieur de l ordinateur Elles permettent a l ordinateur de transmettre et de recevoir des donnees via des protocoles de communication specifiques.

  • SWITCH :

    • C est un composant materiel utilise pourconnecter des appareils sur un reseau local (LAN), il permet au appareils de communiquer entre eux en utilisant des adresses MAC ils sont disponibles en differentes tailles peuvent egalement offrir des fonctionnalites avancees telles que la gestion des VLAN.

  • HUB :

    • C est un composant materiel utilise pour connecter plusieurs appareils sur un reseau local, il hub fonctionne comme un repeteur, il recoit les donnees envoyees par un appareil et les transmete a tous les autres appareils connectes. Les hubs sont remplaces par des switches car ils permettent une communication plus rapide et plus securisee. les hubs sont encore utilises dans des applications specifiques dans des reseaux de petite taille .

2/Le Modele OSI.
    OSI (Open Systems Interconnection)

C est un modele de reference pour les communications de reseau. Il divise les communications en sept couches distinctes, chacune ayant une fonction specifique liee a la communication de donnees entre les appareils de reseau. Les sept couches sont :

  • liaison de donnees.
  • reseau.
  • transport .
  • session.
  • presentation et application.

Le modele OSI est souvent utilise comme reference pour la conception et la mise en oeuvre de reseaux informatiques, bien qu il ne soit pas utilise dans les protocoles de communication de reseau modernes tels que TCP/IP.

3/ Le Modele TCP/IP:

C est un modele de reference pour les communications de reseau utilise dans la conception et la mise en oeuvre des reseaux informatiques modernes. Ce modele divise les communications en quatre couches principales :

  • la couche liaison de donnees.
  • la couche reseau..
  • la couche transport et la couche application.

Contrairement au modele OSI, qui divise les communications en sept couches distinctes, le modele TCP/IP est le modele de reference pour les protocoles de communication de reseau les plus largement utilises, tels que le protocole Internet (IP), le protocole de controle de transmission (TCP) et le protocole de datagramme utilisateur (UDP).

4/L adressage:

Les communications dans un reseau obeissent a des regles :

  • Adressage.
  • Architecture .
  • Protocoles.

    C est quoi un adressage IP :

Une adresse IP c est un numero qui identifie chaque ordinateur dans les reseaux qu il appartient et Il y a deux mode d adressage :

1. ADRESSE IPV4 .
2. ADRESSE IPV6 .

5/ L adresse IPV4 :

Les 4 octets composent une partie pour la Partie reseau et l autre pour la partie hotes .

adressage IPV4 s effectue selon les types suivants :

  • 1) L adresse reseau :

    • tous les bits de la partie hotes=0.

  • 2) L adresse de diffusion :

    • il s agit d une adresse dans laquelle tous les bits de partie hotes=1.

  • 3) L adresse hotes :

    • On attribue les valeurs situes entre l adresse reseau et l adresse de diffusion sur {machines} situe sur cette reseau.

  • 4) Le masque de reseau et de sous reseaux :

    • Tous les bits de la partie reseaux = 1.
